## Welcome

SwaplinQ is a privacy-first, non-custodial smart routing protocol designed to aggregate liquidity and execute instant cross-chain cryptocurrency swaps without ever holding user funds.

The very same infrastructure that powers [swaplinq.com](https://swaplinq.com) is available here as a production-grade REST API. No RPC nodes, no blockchain state management, no transaction complexity. We abstract it all so you can move fast.
